Force Pod Detailed Test

Test Procedure:

  1. Press X to launch Force Pod - should fly forward independently
  2. Press X again to recall - should return to ship
  3. When attached, Force Pod should block enemy bullets (indestructible shield)
  4. When detached, Force Pod should damage enemies on contact
  5. Press C to cycle through weapon types (Red/Blue/Yellow)
  6. Each weapon type has different firing patterns when Force Pod is attached

Wave Cannon Test

Test Procedure:

  1. Hold Z to charge Wave Cannon - charging bar should fill
  2. Normal firing should be disabled while charging
  3. Release Z to fire powerful piercing beam
  4. Beam should penetrate multiple enemies
  5. Beam size/power should scale with charge level
  6. Visual effect should show charging state
